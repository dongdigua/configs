# 我的一些配置
我大概半年前从vim切换到emacs, 用的evil-mode<br>
因为emacs配置感觉比较容易, 而且C-x C-f, C-x C-s和C-g很舒服<br>
但是我最近又从那个200几年的破笔记本上拷出来之前的配置文件, 换回vim<br>
两个原因:<br>
1. vim快, 很快<br>
2. vim的elixir插件的缩进比emacs舒服, emacs必须按enter才能调整那一行的缩进<br>
